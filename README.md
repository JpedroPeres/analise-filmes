# IMDB Movies Analysis (2006–2016)

## Overview
Exploratory data analysis of 1,000 movies from IMDB,
covering the period from 2006 to 2016.
The goal is to uncover patterns in ratings, revenue,
genres, and directors.

## Key Questions Answered
- Which genres have the highest average ratings?
- Does a higher budget lead to a better rating?
- Which directors were most productive in this period?
- How did average revenue evolve over the years?

## Main Insights
- Animation+Drama+Fantasy is the highest-rated genre (8.6)
- Rating and revenue have a weak correlation (0.22)
  — a good movie does not guarantee box office success
- Ridley Scott was the most productive director (8 films)
- Revenue peaked in 2009 and 2012, dropping sharply in 2016

## Technologies
Python | pandas | matplotlib | seaborn

## Dataset
IMDB Movies Dataset — 1,000 movies, 12 columns
Source: Kaggle / LearnDataSci

Open the `.ipynb` file in Google Colab or Jupyter Notebook.
